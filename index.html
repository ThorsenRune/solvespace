Solvespace is a solid model 3D CAD program created by Jonathan Westhues. It uses a parametric design approach driven by geometric solving constraints. Key features are:* Easy to learn with an intuitive user interface – simple but efficient* It can be used to model mechanical behaviour during movements* Solves geometrical constrained problems. * Lends itself to 3D printing. Export obj file for direct import in Slic3 * More robust than Sketch-up for generating manifold models* No set-up needed – runs directly as a small executable and very fast…Extrude, Lathe (spheres), merge and exclude objects…Import componentsBeware of still missing features:* No import*…No advanced geometries (e.g. screw threads). I.e. no ‘follow-me’ as in Sketch-up or macros as in FreeCAD or OpenScadRead more at the original author at http://solvespace.com/index.pl where you find the original program, manuals and exellent tutorials. Please check the forumThis branch has implemented a few enhancements related to the user interface.Summary of changes.
